# First Run Checklist

Use this checklist immediately after installation.

1. Launch Audio Visualizer.
2. Confirm the tray icon appears near the Windows clock.
3. Play audio from any app.
4. Verify the taskbar visualizer animates.
5. Right-click tray icon to test mode and theme changes.

## Common Controls

- Show or hide visualizer
- Mode: Bars, Waveform, Mirror
- Theme and sensitivity
- Quit

## If Something Looks Wrong

- No tray icon: close and relaunch the app from Start Menu.
- No animation: ensure your system output device is active and audio is playing.
- Unexpected behavior: see [../help/TROUBLESHOOTING.md](../help/TROUBLESHOOTING.md).
